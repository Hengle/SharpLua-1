SharpLua is a version of Lua for .NET.
It can run lua files, create .NET objects and call methods on them, and 
supports most lua functions.

Created by mlnlover11 Productions from the LuaInterpreter CodeProject Article.

To use in your projects, just add a reference to this exe and use the 
SharpLua.Interpreter.Run(code[, environment]) or RunFile function.
